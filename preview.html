⚠️ 为什么不能直接双击 index.html？

这是一个现代化的 React 项目，使用了以下技术：

  • ES6 模块： 使用 import/export 语法
  • JSX 语法： React 组件需要编译
  • 模块打包： 需要 Vite 构建工具处理
  • 开发服务器： 提供热重载和实时预览

直接双击 index.html 会遇到 CORS 错误和模块加载问题。

🛠️ 如何预览项目？

方法1：使用开发服务器（推荐）

  1. 在终端中运行：npm run dev
  2. 打开浏览器访问：http://localhost:5173/

方法2：构建静态文件

  1. 运行：npm run build
  2. 运行：npm run preview
  3. 或者直接打开 dist/index.html

📁 项目文件说明

  • index.html - 主HTML模板（需要服务器运行）
  • contact.html - 联系页面（可直接打开）
  • works.html - 作品页面（可直接打开）
  • preview.html - 本说明文件（可直接打开）
