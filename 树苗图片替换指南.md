## 🎨 支持的图片格式

| 格式 | 推荐度 | 说明 |
|------|--------|------|
| **PNG** | ⭐⭐⭐⭐⭐ | 最推荐！支持透明背景，效果最好 |
| **GIF** | ⭐⭐⭐⭐ | 支持动画，非常适合展示树苗成长 |
| **SVG** | ⭐⭐⭐⭐⭐ | 矢量图，任何尺寸都清晰，文件小 |
| **WebP** | ⭐⭐⭐⭐ | 现代格式，文件小，加载快 |
| **JPG/JPEG** | ⭐⭐⭐ | 可用，但不支持透明背景 |

## 📏 推荐的图片尺寸

### 基础要求
- **推荐尺寸**：`300x300` 到 `500x500` 像素
- **最小尺寸**：`200x200` 像素（太小会模糊）
- **最大尺寸**：`800x800` 像素（太大会加载慢）
- **推荐比例**：**1:1（正方形）**
- **文件大小**：每张图片建议不超过 **500KB**

### 为什么推荐正方形？
- 游戏界面为树苗预留的是正方形空间
- 正方形图片显示效果最佳，不会变形
- 如果是长方形，建议留白处理成正方形

## 🛠️ 如何替换图片

### 方法1：使用 public 文件夹（推荐）

#### 步骤1：准备图片
将7个阶段的图片放入 `public` 文件夹，建议命名为：
```
public/
  ├── tree-stage-1.png  （种子阶段：0-1分）
  ├── tree-stage-2.png  （破土而出的小树苗：2-5分）
  ├── tree-stage-3.png  （茁壮的小树苗：6-15分）
  ├── tree-stage-4.png  （有茂密树叶的小树：16-30分）
  ├── tree-stage-5.png  （健壮的大树：31-50分）
  ├── tree-stage-6.png  （开花的大树：51-100分）
  └── tree-stage-7.png  （结满果实的大树：101+分）
```

#### 步骤2：修改代码
打开 `santitg.html`，找到 `treeLevels` 配置（约第759行），修改 `image` 字段：

```javascript
const treeLevels = [
    { minPoints: 0, maxPoints: 1, emoji: '🌱', name: '种子阶段', image: './public/tree-stage-1.png' },
    { minPoints: 2, maxPoints: 5, emoji: '🌿', name: '破土而出的小树苗', image: './public/tree-stage-2.png' },
    { minPoints: 6, maxPoints: 15, emoji: '🌳', name: '茁壮的小树苗', image: './public/tree-stage-3.png' },
    { minPoints: 16, maxPoints: 30, emoji: '🌲', name: '有茂密树叶的小树', image: './public/tree-stage-4.png' },
    { minPoints: 31, maxPoints: 50, emoji: '🎄', name: '健壮的大树', image: './public/tree-stage-5.png' },
    { minPoints: 51, maxPoints: 100, emoji: '🌸', name: '开花的大树', image: './public/tree-stage-6.png' },
    { minPoints: 101, maxPoints: Infinity, emoji: '🍎', name: '结满果实的大树', image: './public/tree-stage-7.png' }
];
```

#### 步骤3：保存并测试
保存文件后，在浏览器中打开游戏，查看效果！

## 🎬 GIF动画建议

如果你想使用GIF动画让树苗"动"起来，请注意以下几点：

### 动画参数建议
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **帧数** | 10-30帧 | 太多会导致文件过大 |
| **循环** | 无限循环 | 让动画持续播放 |
| **背景** | 透明背景 | 与界面更好融合 |
| **动画时长** | 1-3秒 | 一个循环的时长 |
| **文件大小** | <500KB | 保证加载速度 |

### GIF制作工具推荐
- **在线工具**：
  - [Ezgif.com](https://ezgif.com/) - 功能强大的在线GIF编辑器
  - [GifMaker.me](https://gifmaker.me/) - 简单易用的GIF制作工具
  
- **桌面软件**：
  - **Photoshop** - 专业工具，功能全面
  - **GifCam** - 轻量级屏幕录制GIF工具
  - **ScreenToGif** - 免费开源的GIF制作工具

### 动画创意建议
- **种子阶段**：种子微微摇晃
- **破土而出**：小芽向上生长的动画
- **小树苗**：叶子随风摆动
- **茂密小树**：树叶轻轻晃动
- **健壮大树**：树干坚挺，树叶飘动
- **开花大树**：花朵绽放动画
- **结果大树**：果实摇晃，可能有果子掉落

## ⚠️ 常见问题

### Q1: 图片不显示怎么办？
**A:** 检查以下几点：
1. 图片路径是否正确（注意大小写）
2. 图片文件是否确实存在于指定位置
3. 打开浏览器控制台（F12）查看是否有错误信息
4. 确保图片格式浏览器支持

### Q2: 图片显示变形了怎么办？
**A:** 
- 确保使用正方形图片（1:1比例）
- 或者在图片周围添加透明边距，调整为正方形
- 使用图片编辑软件（如Photoshop）调整画布尺寸

### Q3: GIF动画不动了？
**A:**
- 确保GIF文件设置为无限循环
- 检查文件是否损坏，尝试重新导出
- 某些浏览器对过大的GIF可能只显示第一帧

### Q4: 图片加载很慢？
**A:**
- 压缩图片，建议使用 [TinyPNG](https://tinypng.com/) 压缩
- 单张图片不超过500KB
- GIF动画减少帧数或降低画质

### Q5: 能否部分阶段用图片，部分用emoji？
**A:** 
完全可以！只需要在不想替换的阶段保持 `image: ''` 为空即可。
